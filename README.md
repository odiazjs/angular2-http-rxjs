# angular2-http-rxjs		
 - Simple Web application using Angular2 that has a HTTP wrapper using RXJS
 
 ## Upgrading to TSD Typings
 
   - npm uninstall tsd
   - rm -rf typings
   - npm install -g  typings
   - typings init --upgrade
   - rm tsd.json
   - typings install
  
 ## Dependencies 
  
  - npm install
  
 ## Run
  
  - webpack -p
  - npm run dev
