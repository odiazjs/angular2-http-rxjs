## Deprecated tsd 
 
 https://github.com/DefinitelyTyped/tsd/issues/269

Upgrading from tsd to typings

```
$ npm uninstall tsd
$ rm -rf typings
$ npm install -g  typings
$ typings init --upgrade
$ rm tsd.json
$ typings install
```
